Chapter01-01 プロトタイプ・解説

JavaScriptの基本

JavaScriptはオブジェクトが基本であり、扱うのものはほとんどが「オブジェクト」である。

オブジェクトとは

  • 「キー（プロパティ）」と「値」の組み合わせによる連想配列
  • 文字、数、配列等→すべてオブジェクトとして扱う

プロパティとは

  • オブジェクトの設定情報がプロパティ
  • コンストラクタ関数に属していれば独自のプロパティを設定することが可能
    →コンストラクタ関数の宣言時や所属するメソッド内で設定できる

プロトタイプ（prototype）とは

  • JavaScriptに用意されているプロパティのひとつ
  • 同じ処理（メソッド）をグループ化し、再利用や拡張を可能にする

コード解説

function Human(name) {
  this.name = name;
};
Human.prototype.greet = function() {
  console.log("Hello " + this.name);
};
var mike = new Human("Mike");
mike.greet(); //Hello Mike

コンストラクタ関数を定義（オブジェクトの初期化）

function Human(name) {
  this.name = name;
};

function 関数名(引数){ this.変数 = 引数; };

  • 「Human」は任意のコンストラクタ関数名。
  • this.変数のthisは親のコンストラクタ関数名のこと。
    →Humanに所属する変数「this.name」に引数（name）を代入する

複数定義することも可能。

function Human(name, age) {
  this.name = name;
  this.age = age;
};
  • Humanに所属する変数「this.name」に引数（name）を代入する
  • Humanに所属する変数「this.age」に引数（age）を代入する

メソッドを定義する

Human.prototype.greet = function() {
  console.log("Hello " + this.name);
};

コンストラクタ関数.prototype.メソッド名 = function(){処理内容};

  • 「greet」は任意のメソッド名。
  • 同一のコンストラクタ関数を用いて複数のメソッドを定義することができる
  • メソッドは処理内容のグループ化。

オブジェクト変数宣言

var mike = new Human("Mike");

var オブジェクト変数名 = new コンストラクタ関数(引数);

  • 実行コマンドから最初に呼び出される
  • コンストラクタ関数からnew演算子でオブジェクト変数を生成する
  • オブジェクト変数は同じコンストラクタ関数に用意されたメソッドが利用できる

実行コマンド

mike.greet();

オブジェクト変数名.メソッド名();

  • 一番最初に実行する

Ex.コンストラクタ関数とは

  • プログラムのひな形、設計図と呼ばれる。（Java言語ではクラスと呼ぶ）
  • プログラムで必要となる変数を用意するだけで中身は空。
    →独自で作動できない。
  • new演算子で変数を作成すると空の変数へ情報を取り込んで「オブジェクト」を生成する
    →ここで生成された変数が「オブジェクト変数」
    →生成されたオブジェクトが用意されたメソッドを実行したらオブジェクト変数は消える
  • コンストラクタ関数はひたすらオブジェクトを生成することが仕事。
